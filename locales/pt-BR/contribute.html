Contribua

Se você deseja mandar um feedback e/ou reportar um bug, por favor use o Github Issues (você precisa de uma conta no GitHub, que é grátis). Para fazer as coisas mais faceis para todos, por favor, prequise as Issues abertas antes de começar uma noa, assim é evitada a crição de uma Issue inútil e dúplicada.

Todo tipo de ajuda é apreciada, se você for um usuário mais avançado sinta-se livre para submeter uma pull request que você acha que pode melhorar o código ou imprementar novas funções!
