Instructions

  1. Créer un compte Pokemon™ Trainer Club
  2. Se connecter sur l'application Android/iOS pour choisir son starter (vous pouvez utiliser Bluestacks)
  3. Se déconnecter de l'application officielle
  4. Télécharger le ZIP correspondant à votre architecture (ARM pour mobile, x86/x64 pour PC)
  5. Changer vos paramètres de stockage de, stockage sur "Carte SD" à Stockage sur "cet appareil" (Merci à nr23730)
  6. Activer le Portail de gestion des appareils
  7. Aller sur la page APPS du Portail de gestion des appareils
  8. Choisir PokemonGo-UWP_version.appx dans "Install App"
  9. Cliquer sur "Add Dependency" deux fois et ajouter les fichiers qui sont dans le dossier "Dependencies"
  10. Cliquer sur "Go" dans "Deploy" et attendre que le message "Done!" s'affiche
  11. Lancer l'application sur votre appareil
  12. Connectez-vous
  13. Amusez-vous !
