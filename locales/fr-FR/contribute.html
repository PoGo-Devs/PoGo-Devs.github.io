Contribuer

Si vous souhaitez fournir un feedback et/ou signaler un bug, merci d'utiliser ce lien Bug Github (vous aurez besoin d'un compte GitHub gratuit). Pour faciliter le travail de chacun, merci de regarder les problèmes connus avant d'ouvrir un nouveau bug, ceci dans le but d'éviter les doublons inutiles.

Toute aide est appréciée, n'hésitez pas à soumettre vos changements (pull request) si vous pensez pouvoir améliorer le code ou ajouter des fonctionnalitées!
