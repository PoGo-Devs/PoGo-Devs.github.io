Contribueren

Als je feedback en/of een fout wilt rapporteren, a.u.b gebruik dan Github Issues (Je hebt een gratis GitHub account nodig). Om dingen makkelijker te maken, a.u.b zoek dan eerst naar het probleem wat u heeft om duplicaties te voorkomen.

Elke vorm van hulp is gewaardeerd, voel je vrij om een pull request te maken als je denkt dat je mijn code kunt verbeteren of nieuwe dingen kunt toevoegen!
